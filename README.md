# Image Manipulator

## Description
This program is able to filters such as grayscale, sharpen, blur, and more to images. When saving an image, the program is able
to convert the image to a multitude of formats. You can save any image as the following formats: png, jpg, msoe, and bmsoe.
The msoe and bmsoe are proprietary image formats made for this project. This project was made with java and javafx.

## How to use
- Click open image (some base images are provided)
- Add filters to the image
- The reload button resets the image to its original state.
- Click save button to save an image
